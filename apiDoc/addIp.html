API Документація

Ендпоінт: POST /api/addIp.html

Опис: Цей ендпоінт дозволяє додати нову IP-адресу до списку довірених адрес у базі даних ip_database.

Формат запиту

Метод запиту: POST

Тип вмісту: application/json

Тіло запиту:

{
    "api_key": "your_api_key",
    "ip": "192.168.1.1"
}
  • api_key (обов'язкове): API ключ, який вже був згенерований і збережений у базі даних api_keys.
  • ip (обов'язкове): IP-адреса у форматі IPv4, яку потрібно додати до довірених.

Відповідь:

Тип вмісту: application/json

Успішне виконання:

{
    "status": "success",
    "message": "IP address has been added successfully."
}

Помилка:

{
    "status": "error",
    "message": "Invalid API key or IP address."
}
